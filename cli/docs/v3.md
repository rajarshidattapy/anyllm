# PRD: anyllm Slash Commands — init, pack, push, repack

**Status:** Ready to implement  
**Scope:** 4 global slash command files + anyllm CLI additions (`push`, `repack` subcommands)  
**Token budget:** 0 Claude tokens per invocation (hard requirement on all four)

---

## Commands Overview

| Command | What it does |
|---|---|
| `/anyllm-init` | Initialize `.anyllm/` in current project |
| `/anyllm-pack` | Ingest current session → distill → merge into `current.md` |
| `/anyllm-push` | Paste briefing into OpenAI Codex and press Send — silently, no terminal output |
| `/anyllm-repack` | Re-ingest turns since last pack, merge delta into `current.md` |

## `/anyllm-push`

### What it does

Reads `.anyllm/current.md`, composes the briefing, then:

1. Opens (or focuses) the Codex browser tab
2. Clicks the input field
3. Pastes the briefing
4. Presses Enter / clicks Send

No briefing text is printed to terminal. No clipboard contents are shown. Claude Code sees nothing — the push happens entirely in a subprocess.

### Platform implementations

The `anyllm push` CLI subcommand must detect platform and use the right automation path:

**Linux (X11)**
```
xdotool search --name "Codex" windowfocus
xdotool type --clearmodifiers -- "$(anyllm compose)"
xdotool key Return
```

**Linux (Wayland)**
```
# xdotool doesn't work on Wayland
# Use ydotool (requires ydotoold daemon running)
ydotool type -- "$(anyllm compose)"
ydotool key Return
```

**macOS**
```applescript
tell application "System Events"
  set the clipboard to "<briefing>"
  tell process "Arc" -- or "Google Chrome", "Safari"
    keystroke "v" using command down
    key code 36 -- Return
  end tell
end tell
```

**Windows (WSL)**
```powershell
Add-Type -AssemblyName System.Windows.Forms
[System.Windows.Forms.Clipboard]::SetText("<briefing>")
[System.Windows.Forms.SendKeys]::SendWait("^v{ENTER}")
```

### Browser targeting

Codex may be open in any tab. The push command must find it reliably.

Detection order:
1. Check `config.yaml` for `push.browser` and `push.codex_url` (user-specified, highest priority)
2. Search open windows/tabs for a title matching `codex` (case-insensitive)
3. If not found — open `https://codex.openai.com` in the default browser, wait 3 seconds for load, then inject

`config.yaml` additions:
```yaml
push:
  browser: auto          # auto | chrome | arc | firefox | safari
  codex_url: https://codex.openai.com
  send_delay_ms: 500     # wait after focus before typing (default 500ms)
  open_if_missing: true  # open Codex tab if not found
```

### Silence requirement

`anyllm push` must not print the briefing to stdout at any point. Internal flow:

```
anyllm push
  └─ compose briefing → held in memory / temp file (never stdout)
  └─ platform detect → choose automation path
  └─ focus Codex tab
  └─ inject text + send
  └─ print one confirmation line: "✓ pushed to Codex"
```

The single confirmation line is safe — it's not the briefing content, it doesn't pollute context.

### CLI subcommand to implement: `anyllm push`

**New file:** `src/anyllm/push.py`

```python
def push(target="codex", config=None):
    briefing = compose_briefing(config)   # returns string, never prints
    platform = detect_platform()          # linux_x11 | linux_wayland | macos | windows
    injector = get_injector(platform)
    injector.focus_target(target, config)
    injector.inject_and_send(briefing, delay_ms=config.push.send_delay_ms)
    print("✓ pushed to Codex")
```

Dependencies to add:
- Linux: `xdotool` (X11), `ydotool` (Wayland) — system packages, not pip
- macOS: `osascript` — built in
- Windows: `pywin32` — pip

## `/anyllm-repack`

### The problem it solves

You ran `/anyllm-pack` at turn 40. You then did turns 41, 42, 43 (forgot to pack). Now you're about to switch to Codex. Those 3 turns have decisions, file changes, a next step — none of it is in `current.md`.

`/anyllm-repack` ingests only turns 41–43, distills just that delta, and merges it into the existing `current.md`. The result is `current.md` that reflects everything up to turn 43.

### How it finds the leftover turns

`index.json` already records every pack operation with a timestamp:

```json
{
  "sessions": [
    {
      "session_id": "abc123",
      "packed_at": "2026-06-29T14:00:00Z",
      "turn_count": 40,
      "last_turn_ts": "2026-06-29T13:58:22Z"
    }
  ]
}
```

`anyllm repack` reads `index.json`, finds `last_turn_ts` of the most recent pack, then re-ingests the transcript filtered to turns after that timestamp.

### Ingestor change required

`ClaudeCodeIngestor` needs a `since_ts` parameter:

```python
def ingest(self, session_id=None, since_ts=None) -> NormalizedTranscript:
    turns = self._parse_jsonl(session_id)
    if since_ts:
        turns = [t for t in turns if t["ts"] > since_ts]
    return NormalizedTranscript(turns=turns, ...)
```

If `since_ts` filters down to 0 turns → repack exits early with: `"Nothing to repack — no new turns since last pack."`

### Distillation of a delta

The distiller receives a small transcript (2–5 turns typically). The system prompt for repack uses a different prompt variant: `distiller/prompts/v1-delta.md`

`v1-delta.md` must instruct the LLM:
- This is a partial transcript, not a full session
- Focus only on what changed: new decisions, new files touched, updated next step
- Do not re-summarize what was already done — the merge engine handles continuity
- Output the same snapshot format as `v1.md` but sections may be sparse

### Merge behavior

Repack feeds into the existing `MergeEngine.merge()` — no changes needed there. The delta snapshot is merged into `current.md` exactly like a normal pack. Stale/orphaned/confirmed logic applies.

`index.json` gets a new entry tagged `"type": "repack"` so the log shows the distinction:

```json
{ "session_id": "abc123", "packed_at": "...", "type": "repack", "turns_ingested": 3, "since_ts": "..." }
```

### CLI subcommand to implement: `anyllm repack`

```python
def repack(source=None, config=None):
    last_pack = storage.get_last_pack_entry()   # reads index.json
    if not last_pack:
        print("No previous pack found. Run /anyllm-pack first.")
        return

    since_ts = last_pack["last_turn_ts"]
    transcript = ingestor.ingest(since_ts=since_ts)

    if not transcript.turns:
        print("Nothing to repack — no new turns since last pack.")
        return

    print(f"Repacking {len(transcript.turns)} missed turns since {since_ts}...")
    snapshot = distiller.distill(transcript, prompt_version="v1-delta")
    storage.write_snapshot(snapshot, tag="repack")
    merge_result = merger.merge(storage.read_current(), snapshot)
    storage.write_current(merge_result.merged_md)
    storage.append_index_entry(transcript, tag="repack", since_ts=since_ts)
    print(f"✓ repack done — {merge_result.added} added, {merge_result.confirmed} confirmed")
```
